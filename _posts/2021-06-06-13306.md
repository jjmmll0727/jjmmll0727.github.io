```python
city_count = int(input())
value_sum = 0
roads = list(map(int, input().split()))
values = list(map(int, input().split()))
min_val = values[0]

for i in range(city_count-1):
    if min_val > values[i]:
        min_val = values[i]
    value_sum += min_val * roads[i]
print(value_sum)
```
